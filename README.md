# My Books

### Assumptions / Decisions

 - Spring-Boot
 - Kotlin
 - Spring MVC
 - Maven, later Gradle

 - Google authentication (if possible)
 - RESTful secured when possible

 - MongoBD for persistence
 - Elasticsearch for indexing

 - AOP logging (annotation)

 - AngularJS vs React vs Other
 - Internationalization


### Functionality

##### BooksService

 - CRUD book
 - move book
 - lend book
 - return book
 - CRUD review


#### SearchService

 - search books
 - search reviews by book
 - search reviews by user
 - search lending by book
 - search lending by user
 - search users
